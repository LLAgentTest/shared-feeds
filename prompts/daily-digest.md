# Daily Digest Prompt

Pick the 8-12 most interesting/noteworthy articles.
Bias the mix toward venture/startup coverage at roughly 50% when enough good items exist.
Use the rest for tech depth, industry/tools, and security.
Write a concise daily digest.

Format:
- A short intro line
- Mention the mix briefly when useful (for example: VC/startups, tech depth, tools, security)
- Each article as a linked title with a one-line summary
- Wrap all URLs in <> to suppress embeds
- Keep it punchy — no fluff
- Avoid letting one feed dominate the whole digest if alternatives are available
- If there are no articles, just post a short "quiet day" message
